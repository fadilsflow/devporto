
# Developer Portofolio Starter
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/fadilsflow/devporto)

Built with next.js, [shadcn/ui](https://ui.shadcn.com/), deployed on Vercel.

## Features

- Setup only takes a few minutes by editing the [single config file](./app/data.ts)
- Built using Next.js 14, React, Typescript, Shadcn/UI, TailwindCSS, Framer Motion
- Includes a blog 
- rss
- Project section and page exmpalin
- Github Stats section
- links three page
- Responsive for different devices


## Getting Started Locally

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/fadilsflow/devporto
   ```

2. Move to the cloned directory

   ```bash
   cd portfolio
   ```

3. Install dependencies:

   ```bash
   npm install
   # or
   bun install
   ```

4. Start the local Server:

   ```bash
   npm run dev
   # or
   bun dev
   ```

5. Open the [Config file](./app/data.ts) and make changes
